# Smart-Safety-Planner-for-Women
A simple full-stack web application that helps women students plan their daily schedule more safely using smart time-based risk analysis and emergency features.

Problem Statement
Women students often travel for college, tuition, or personal work. However, while planning their day, safety risks like late-night travel, unsafe outing times, and lack of emergency support are usually ignored.
Existing planners do not provide safety-based suggestions. This project focuses on solving that gap.

Solution
This project introduces a smart safety planner that:
Detects risky task timings
Generates a safety score
Suggests a safer daily schedule
Allows users to store emergency contacts
Provides a check-in safety reminder.

Key Features
Smart daily planner
Safety score (0–100)
Risk detection based on task timing
Emergency contact support
Clean and simple user interface.

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: JSON (LowDB)
